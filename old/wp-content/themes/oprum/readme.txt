== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Quick Start ==

1. First step: Settings > Reading: Front page displays - select Your latest posts

2. If you want to display the content of any page on the Home: Appearance > Theme Options: Feature Page

3. Choose your color, upload the logo image and other, go to Appearance > Customize

4. Theme supports 2 menus: Primary and Social icons. Create Menus, go to Appearance > Menus
*Social Menu location placed in the right footer

5. Define sticky posts (Posts > All Posts:  Quick Edit - check Make this post sticky) for display on Home
*Use Featured image meta box Post Edit

6. The Theme includes Contextual Help. Use the contextual help (click on the tab Help at the top right)

**Tips**

Using Customizer set your color, upload a logo (face - for personal) image, upload a background image (or select any color), upload the image (or select any color) header and other settings.

As the Front page displays select Your latest posts, go to Customize > section Static Front Page: select Your latest posts. You can display the contents of any page on the home by using theme Options Feature Page.

In the section of Home Tagline can be used shortcode. For example, instead of heading to place the slider using shortcode slideshow plugin Soliloquy Sliders.

If you want to display the posts without the sidebar just leave blank (no widgets) sidebar posts.

This theme is ready for the Woocommerce plugin. Can create a store and sell your products, e-books.
